Salam Sadideeee salam khubi_#

This is the team documentation for the ExecutableBooksProject, an international collaboration to build open source tools that facilitate publishing computational narratives using the Jupyter ecosystem. Below are two marquee projects that we steward.

Jupyter Book

A command-line interface for building beautiful, publication-quality books and documents from computational content.

Jupyter Book

MyST Markdown

An extensible, semantic, and community-driven flavor of markdown designed for scientific and computational narratives.

MyST Markdown
